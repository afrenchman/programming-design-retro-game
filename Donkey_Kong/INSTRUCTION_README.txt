To run this game
>> python game_mk.py
Controller used for Testing:
Logitech G F310 Wired Gamepad Controller Console Like Layout 4 Switch D-Pad PC - Blue/Black

Gameplay Mechanics:
Left D-Pad: Mario moves left
Right D-Pad: Mario moves right 
Up D-pad: Mario moves upstairs (if ladders are not broken)
Down D-pad: Mario moves downstairs (if ladders are not broken)
Red B Button: Mario Jumps
Blue X Button: Exits back to main menu
